# Power-BI-Dashboard
📌 Project Overview

This project is an interactive Sales Performance Dashboard built using Microsoft Power BI. The dashboard provides a comprehensive overview of business performance by visualizing sales, profit, customer segments, payment modes, product categories, and regional sales. It helps stakeholders analyze business trends, identify top-performing products, and make data-driven decisions.

🚀 Features
📈 Total Sales and KPI Cards
💰 Average Order Value (AOV)
🌍 Regional Sales Analysis
📦 Sales by Category & Sub-Category
👥 Sales by Customer Segment
💳 Sales by Payment Mode
📅 Monthly & Yearly Sales Trends
📊 Monthly & Yearly Profit Analysis
🗺️ Geographic Sales & Profit Distribution (Map Visualization)
Interactive filters and drill-down analysis
